# Contributing to AI Agent Card Standard

Thank you for your interest!

## Ways to Contribute
- Report issues or suggest improvements to the schema
- Provide real-world populated examples
- Propose mappings to additional frameworks
- Submit pull requests for documentation or schema enhancements

## Process
1. Open an issue describing the change
2. For substantial changes, discuss in the issue first
3. Ensure any new example passes validation
4. Keep focus on transparency and human agency

We aim for a collaborative, practical standard — not perfection on first try.
